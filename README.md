# Mini TeamFlow

Flutter + Laravel ile geliştirilecek şirket içi görev, check-in, takvim ve bildirim uygulaması.

## Kararlar

- Kullanıcı oluşturma: İlk admin manuel/seed ile oluşturulur, sonra admin çalışan ekler.
- Ekip yapısı: Tek ekip, departman yok.
- Görev oluşturma: Admin ve çalışan görev oluşturabilir.
- Takvim: Çalışan kendi kişisel etkinliğini ekleyebilir.
- Check-in: Zorunlu değil; admin kimin yapmadığını görebilir.
- Dil: Türkçe.
- Tasarım: Modern startup tarzı.
- Admin panel: Şimdilik yok, ileride eklenebilir.
- Bildirim: MVP içinde yapılacak.
- Kullanım hedefi: Gerçek kullanıma yakın, şirket içi in-house kullanıma uygun.

## Teknoloji

- Mobil: Flutter
- Backend: Laravel REST API
- Auth: Laravel Sanctum
- Database: MySQL veya PostgreSQL

## Fazlar

1. Temel kurulum + auth + rol yapısı ✅
2. Görev sistemi ✅
3. Günlük check-in ✅
4. Takvim/etkinlik ✅
5. Dashboard/raporlar ✅
6. UI/UX iyileştirme + bildirimler ✅
